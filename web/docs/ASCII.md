# ASCII Node

This node transforms images into ASCII art-like representations.

## Parameters

- **image**: Input image to process
- **background**: Background color (default: #080c37)
- **bgGradient**: Enable background gradient (default: False)
- **bgSaturation**: Background saturation (0-100, default: 60)
- **fontColor**: Primary font color (default: #c7205b)
- **fontColor2**: Secondary font color (default: #00ff61)
- **fontSizeFactor**: Font size factor (0.1-10.0, default: 3.0)
- **resolution**: Output resolution (default: 137)
- **threshold**: Luminance threshold (0-255, default: 0)
- **invert**: Invert colors (default: True)
- **randomness**: Randomness factor (0-100, default: 15)
- **textType**: Type of text input (default: Random Text)
- **textInput**: Custom text input (default: pxlpshr)
